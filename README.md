## Description
Banking application :- A typical Banking application example built with nestjs as its backend, Graphql for its api end-point, Mongodb for its database to store users information and banking transaction informations

## Prerequisite

- Node.js installed
- Yarn installed
- Basic javascript and typescript knowledge
- Express.js
- Basic undestanding of graphql and how it works
- Apollo-graphql
- Use of databases
- Understand mongodb usage and connection to node.js app

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
